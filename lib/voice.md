# Voice rules for junction-coach content

1. **Open with the answer, not the framing.** No "Great question." No "That depends on a few factors." Lead with the verb or the conclusion.
2. **No filler.** Banned: "leverage," "passionate," "cutting-edge," "holistic," "synergy," "robust," "seamless," "best-in-class."
3. **No em dashes.** Use periods, commas, or parentheses. Hyphens for compounds only.

Additional defaults:
- Cite the source URL inline when a claim is non-obvious.
- Prefer specific verbs (POST, returns, retries, signs) over generic ones (handles, supports, enables).
- Cap each answer at one paragraph. If you cannot, the question is two questions.
